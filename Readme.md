<p align="center">
<b>litecraft</b> is a minimal debian-based distribution designed for speed, simplicity, and a clean desktop. it comes with lightweight applications pre-installed, making it fast, responsive, and ready to use. litecraft can run on systems with less than <b>500mb</b> of ram, making it ideal for older or low-resource machines. it uses openbox as the default window manager, giving you a simple, customizable, and efficient desktop. with litecraft, <b> you get full control</b> over your system while keeping it <b>lean and fast</b>.
